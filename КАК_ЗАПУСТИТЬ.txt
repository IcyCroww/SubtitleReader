🚀 БЫСТРЫЙ ЗАПУСК:

1. Откройте папку:
   SubtitleReader\bin\Release\net9.0-windows\

2. Запустите файл:
   SubtitleReader.exe

3. Готово! Программа запущена.

═══════════════════════════════════════════════════════════════

✅ ЧТО УЖЕ НАСТРОЕНО:

✓ Языковые файлы для OCR (русский + английский)
✓ Все необходимые библиотеки
✓ Готовая к работе программа

═══════════════════════════════════════════════════════════════

📝 КАК ПОЛЬЗОВАТЬСЯ:

1. Нажмите "🎯 Выделить область на экране"
2. Выделите область с текстом (зажмите левую кнопку мыши)
3. Текст автоматически распознается
4. Нажмите "▶ Читать" для озвучивания

═══════════════════════════════════════════════════════════════

🎮 ДЛЯ ИГР:

1. Создайте несколько областей для разных частей экрана
2. Включите "Авто-чтение" для нужных областей
3. Нажмите "🔄 Старт мониторинга"
4. Программа будет автоматически читать новый текст
5. Сохраните настройки как заготовку для игры

═══════════════════════════════════════════════════════════════

⚙️ НАСТРОЙКИ:

• Выбор голоса для озвучивания
• Громкость и скорость чтения
• Горячие клавиши (F9, F10, F11)
• Интервал мониторинга областей

═══════════════════════════════════════════════════════════════

💡 ПОЛЕЗНЫЕ ФУНКЦИИ:

• 👁️ Overlay - показывает области поверх других окон
• 📷 Захватить текст - распознать текст в области
• 📋 Копировать - скопировать текст в буфер обмена
• 💾 Заготовки - сохранить настройки для разных игр

═══════════════════════════════════════════════════════════════

❓ ПРОБЛЕМЫ?

Если программа не запускается:
• Установите .NET 9.0 Runtime с сайта Microsoft
• Запустите от имени администратора
• Проверьте антивирус

Если OCR не работает:
• Файлы rus.traineddata и eng.traineddata должны быть
  в папке tessdata рядом с SubtitleReader.exe
